# My Channel Website

This is a beautiful, responsive website for your channel, designed with Tailwind CSS to showcase your content and engage with your audience.

## Features

- Modern, responsive design using Tailwind CSS
- Beautiful UI with gradient backgrounds and smooth transitions
- Featured content section to highlight your best videos
- About section to tell your story
- Subscribe section to grow your audience
- Clean, professional footer with navigation

## Quick Start

The website is built with React and Tailwind CSS. You can run it locally with:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Static Version

A static version of the website is available in the `dist` folder that can be deployed directly to any web server without build tools.

## Customization

You can easily customize this website to match your channel's brand:

- Update the color scheme by modifying Tailwind classes
- Replace placeholder content with your actual videos and information
- Add your channel logo and images
- Modify the navigation to match your content structure

## Technologies Used

- React

- Tailwind CSS
